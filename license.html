Used libraries¶

Some parts of the code were copied and adapted from the Object-Oriented Micromagnetic Framework (OOMMF), which is released as public domain code by Michael Donahue and Don Porter:

  • the implementation of Newells formulas in src/magneto/mmm/demag/demag_coeff.h
  • the cubic anisotropy field computation in src/magneto/mmm/anisotropy/anisotropy_cpu.cpp

The VTK output routines were adapted from the PyEVtk library at https://bitbucket.org/pauloh/pyevtk by Paulo A. Herrera. The copyright notice of the PyEVtk package is shown below:
